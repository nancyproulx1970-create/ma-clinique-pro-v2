# Guide — Création du projet Supabase Staging

## Objectif

Créer un environnement **staging** Supabase isolé de la production,  
avec les tables principales, RLS activé, et l'Edge Function `generate-note` déployée.

**Priorité absolue : sécuriser la clé OpenAI avant d'avancer sur d'autres fonctionnalités.**

---

## Étape 1 — Créer le projet Supabase Staging

### Via le tableau de bord Supabase

1. Aller sur [https://supabase.com/dashboard](https://supabase.com/dashboard)
2. Cliquer **"New project"**
3. Remplir :
   - **Name** : `ma-clinique-pro-v2-staging`
      - **Database Password** : générer un mot de passe fort (le conserver dans votre gestionnaire de mots de passe)
         - **Region** : `Canada (Central)` — important pour conformité LPRPDE/Québec
            - **Plan** : Free (pour staging)
            4. Cliquer **"Create new project"** — attendre ~2 minutes

            ### Noter les informations du projet staging

            Après création, aller dans `Settings → API` et noter :
            - **Project URL** : `https://VOTRE_ID_STAGING.supabase.co`
            - **Project ID** : `VOTRE_ID_STAGING`
            - **anon key** : (clé publique — peut être dans FlutterFlow)
            - **service_role key** : (clé privée — JAMAIS dans FlutterFlow, seulement dans les Edge Functions)

            ## Étape 2 — Créer les tables principales

            Dans le tableau de bord Supabase : `SQL Editor → New query`

            Copier et exécuter ce SQL dans l'ordre :

            ### 2.1 — Table `patients`

            ```sql
            CREATE TABLE patients (
              id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                created_at TIMESTAMPTZ DEFAULT now(),
                  updated_at TIMESTAMPTZ DEFAULT now(),

                    -- Identité
                      nom TEXT NOT NULL,
                        prenom TEXT NOT NULL,
                          date_naissance DATE,
                            telephone TEXT,
                              email TEXT,
                                adresse TEXT,
                                  ville TEXT,
                                    code_postal TEXT,
                                      province TEXT DEFAULT 'QC',

                                        -- Médical
                                          antecedents TEXT,
                                            allergies TEXT,
                                              medicaments TEXT,
                                                medecin_referent TEXT,
                                                  notes_generales TEXT,

                                                    -- Gestion
                                                      actif BOOLEAN DEFAULT true,
                                                        created_by UUID REFERENCES auth.users(id) ON DELETE SET NULL,
                                                          clinique_id UUID  -- pour multi-cliniques futur
                                                          );

                                                          -- Index de recherche
                                                          CREATE INDEX patients_nom_prenom_idx ON patients(nom, prenom);
                                                          CREATE INDEX patients_created_by_idx ON patients(created_by);
                                                          ```

                                                          CREATE TABLE rendez_vous (
                                                            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                                                              created_at TIMESTAMPTZ DEFAULT now(),
                                                                updated_at TIMESTAMPTZ DEFAULT now(),

                                                                                        type_soin TEXT NOT NULL DEFAULT 'Soins podologiques à domicile',
                                                                                          statut TEXT NOT NULL DEFAULT 'planifie'
                                                                                              CHECK (statut IN ('planifie', 'en_cours', 'complete', 'annule', 'reporte')),

                                                                                                -- Localisation visite
                                                                                                  adresse_visite TEXT,
                                                                                                    ville_visite TEXT,
                                                                                                      code_postal_visite TEXT,
                                                                                                      
                                                                                                        -- Note clinique
                                                                                                          observations TEXT,
                                                                                                            note_clinique TEXT,

                                                                                                                -- Facturation
                                                                                                                  montant_honoraires DECIMAL(10,2),
                                                                                                                    tps_applicable BOOLEAN DEFAULT true,
                                                                                                                      tvq_applicable BOOLEAN DEFAULT true,

                                                                                                                                        source TEXT DEFAULT 'manuel'

                                                                                                                                            CREATE INDEX rdv_clinicienne_idx ON rendez_vous(clinicienne_id);
                                                                                                                                            CREATE INDEX rdv_date_idx ON rendez_vous(date_debut);

                                                                                                                                                                                                                                                ## Étape 3 — Activer Row Level Security (RLS)
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                **RLS doit être activé sur TOUTES les tables. Sans RLS, les données sont accessibles à tous.**
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                ```sql
                                                                                                                                                                                                                                                -- Activer RLS sur toutes les tables
                                                                                                                                                                                                                                                ALTER TABLE patients ENABLE ROW LEVEL SECURITY;
                                                                                                                                                                                                                                                ALTER TABLE rendez_vous ENABLE ROW LEVEL SECURITY;
                                                                                                                                                                                                                                                ALTER TABLE photos_visite ENABLE ROW LEVEL SECURITY;
                                                                                                                                                                                                                                                ALTER TABLE cliniciennes ENABLE ROW LEVEL SECURITY;
                                                                                                                                                                                                                                                ALTER TABLE ia_logs ENABLE ROW LEVEL SECURITY;
                                                                                                                                                                                                                                                ```
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                ### Politiques RLS — Table `patients`
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                ```sql
                                                                                                                                                                                                                                                -- Une clinicienne voit uniquement ses propres patients
                                                                                                                                                                                                                                                CREATE POLICY "patients_select_own" ON patients
                                                                                                                                                                                                                                                  FOR SELECT USING (auth.uid() = created_by);
                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                  CREATE POLICY "patients_insert_own" ON patients
                                                                                                                                                                                                                                                    FOR INSERT WITH CHECK (auth.uid() = created_by);
                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                    CREATE POLICY "patients_update_own" ON patients
                                                                                                                                                                                                                                                      FOR UPDATE USING (auth.uid() = created_by);
                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                      -- Pas de DELETE — désactiver un patient (actif = false) plutôt que supprimer
                                                                                                                                                                                                                                                      ```
                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                      ### Politiques RLS — Table `rendez_vous`
                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                      ```sql
                                                                                                                                                                                                                                                      -- Une clinicienne voit uniquement ses propres RDV
                                                                                                                                                                                                                                                      CREATE POLICY "rdv_select_own" ON rendez_vous
                                                                                                                                                                                                                                                        FOR SELECT USING (auth.uid() = clinicienne_id);
                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                        CREATE POLICY "rdv_insert_own" ON rendez_vous
                                                                                                                                                                                                                                                          FOR INSERT WITH CHECK (auth.uid() = clinicienne_id);
                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                          CREATE POLICY "rdv_update_own" ON rendez_vous
                                                                                                                                                                                                                                                            FOR UPDATE USING (auth.uid() = clinicienne_id);
                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                            -- Annulation uniquement (pas suppression physique)
                                                                                                                                                                                                                                                            CREATE POLICY "rdv_delete_own" ON rendez_vous
                                                                                                                                                                                                                                                              FOR DELETE USING (auth.uid() = clinicienne_id AND statut = 'annule');
                                                                                                                                                                                                                                                              ```
                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                              ### Politiques RLS — Table `photos_visite`
                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                              ```sql
                                                                                                                                                                                                                                                              -- Via le RDV auquel la photo appartient
                                                                                                                                                                                                                                                              CREATE POLICY "photos_select_via_rdv" ON photos_visite

                                                                                                                                                                                                                                                                    EXISTS (

                                                                                                                                                                                                                                                                                            )
                                                                                                                                                                                                                                                                                              );
                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                              CREATE POLICY "photos_insert_via_rdv" ON photos_visite
                                                                                                                                                                                                                                                                                                FOR INSERT WITH CHECK (
                                                                                                                                                                                                                                                                                                    EXISTS (
                                                                                                                                                                                                                                                                                                          SELECT 1 FROM rendez_vous r
                                                                                                                                                                                                                                                                                                                WHERE r.id = rendez_vous_id
                                                                                                                                                                                                                                                                                                                        AND r.clinicienne_id = auth.uid()
                                                                                                                                                                                                                                                                                                                            )
                                                                                                                                                                                                                                                                                                                              );
                                                                                                                                                                                                                                                                                                                              ```

                                                                                                                                                                                                                                                                                                                                      ## Étape 4 — Triggers utiles
                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                      ```sql
                                                                                                                                                                                                                                                                                                                                      -- Trigger : mettre à jour updated_at automatiquement
                                                                                                                                                                                                                                                                                                                                      CREATE OR REPLACE FUNCTION update_updated_at_column()
                                                                                                                                                                                                                                                                                                                                      RETURNS TRIGGER AS $$
                                                                                                                                                                                                                                                                                                                                      BEGIN
                                                                                                                                                                                                                                                                                                                                        NEW.updated_at = now();
                                                                                                                                                                                                                                                                                                                                          RETURN NEW;
                                                                                                                                                                                                                                                                                                                                          END;
                                                                                                                                                                                                                                                                                                                                          $$ LANGUAGE plpgsql;
                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                          CREATE TRIGGER patients_updated_at
                                                                                                                                                                                                                                                                                                                                            BEFORE UPDATE ON patients
                                                                                                                                                                                                                                                                                                                                              FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();
                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                              CREATE TRIGGER rdv_updated_at
                                                                                                                                                                                                                                                                                                                                                BEFORE UPDATE ON rendez_vous
                                                                                                                                                                                                                                                                                                                                                  FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();
                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                  CREATE TRIGGER cliniciennes_updated_at
                                                                                                                                                                                                                                                                                                                                                    BEFORE UPDATE ON cliniciennes
                                                                                                                                                                                                                                                                                                                                                      FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();
                                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                                      -- Trigger : créer le profil clinicienne après inscription Auth
                                                                                                                                                                                                                                                                                                                                                      CREATE OR REPLACE FUNCTION handle_new_user()
                                                                                                                                                                                                                                                                                                                                                      RETURNS TRIGGER AS $$
                                                                                                                                                                                                                                                                                                                                                      BEGIN
                                                                                                                                                                                                                                                                                                                                                        INSERT INTO cliniciennes (id, email, nom, prenom)
                                                                                                                                                                                                                                                                                                                                                          VALUES (
                                                                                                                                                                                                                                                                                                                                                              NEW.id,
                                                                                                                                                                                                                                                                                                                                                                  NEW.email,
                                                                                                                                                                                                                                                                                                                                                                      COALESCE(NEW.raw_user_meta_data->>'nom', 'Nouveau'),
                                                                                                                                                                                                                                                                                                                                                                          COALESCE(NEW.raw_user_meta_data->>'prenom', 'Utilisateur')
                                                                                                                                                                                                                                                                                                                                                                            );
                                                                                                                                                                                                                                                                                                                                                                              RETURN NEW;
                                                                                                                                                                                                                                                                                                                                                                              END;
                                                                                                                                                                                                                                                                                                                                                                              $$ LANGUAGE plpgsql SECURITY DEFINER;
                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                              CREATE TRIGGER on_auth_user_created
                                                                                                                                                                                                                                                                                                                                                                                AFTER INSERT ON auth.users
                                                                                                                                                                                                                                                                                                                                                                                  FOR EACH ROW EXECUTE FUNCTION handle_new_user();
                                                                                                                                                                                                                                                                                                                                                                                  ```

                                                                                                                                                                                                                                                                                                                                                                                  ## Étape 5 — Configurer le Storage (photos)
                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                  Dans le tableau de bord Supabase : `Storage → New bucket`
                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                  - **Nom** : `photos-visites`
                                                                                                                                                                                                                                                                                                                                                                                  - **Public** : NON (privé)
                                                                                                                                                                                                                                                                                                                                                                                  - **Allowed MIME types** : `image/jpeg, image/png, image/webp`
                                                                                                                                                                                                                                                                                                                                                                                  - **Max file size** : 10 MB
                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                  ### Politique Storage
                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                  ```sql
                                                                                                                                                                                                                                                                                                                                                                                  -- Dans Storage policies, pour le bucket photos-visites :
                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                  -- SELECT : une clinicienne accède uniquement à ses photos
                                                                                                                                                                                                                                                                                                                                                                                  CREATE POLICY "photos_storage_select" ON storage.objects
                                                                                                                                                                                                                                                                                                                                                                                    FOR SELECT USING (
                                                                                                                                                                                                                                                                                                                                                                                        bucket_id = 'photos-visites'
                                                                                                                                                                                                                                                                                                                                                                                            AND auth.uid()::TEXT = (storage.foldername(name))[1]
                                                                                                                                                                                                                                                                                                                                                                                              );
                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                              -- INSERT
                                                                                                                                                                                                                                                                                                                                                                                              CREATE POLICY "photos_storage_insert" ON storage.objects
                                                                                                                                                                                                                                                                                                                                                                                                FOR INSERT WITH CHECK (
                                                                                                                                                                                                                                                                                                                                                                                                    bucket_id = 'photos-visites'
                                                                                                                                                                                                                                                                                                                                                                                                        AND auth.uid()::TEXT = (storage.foldername(name))[1]
                                                                                                                                                                                                                                                                                                                                                                                                          );
                                                                                                                                                                                                                                                                                                                                                                                                          ```
